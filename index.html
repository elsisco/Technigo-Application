Web server

A web server is software and hardware that uses HTTP and other protocols to respond to client requests made over the World Wide Web. The main job of a server is to display website content through storing, processing and delivering webpages to users.

A web server can be a single computer but high-traffic websites typically run web servers on fleets of computers designed to handle large numbers of requests. Web server software controls how a user accesses hosted files, while web server hardware is connected to the internet and allows data to be exchanged with other connected devices.
